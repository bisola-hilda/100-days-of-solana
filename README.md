## Day 5 – Comparing Wallet Types

### What I Learned

Today I compared three different wallet types:

- CLI Wallet
- Browser Wallet
- Mobile Wallet

I learned that all wallets manage the same cryptographic keypair but provide different levels of security, convenience, and recovery options.

### Reflection

- CLI wallets are excellent for development and automation.
- Browser wallets are ideal for interacting with decentralized applications.
- Mobile wallets provide stronger security through encrypted storage and biometrics.
- Hardware wallets remain the safest choice for storing large amounts of cryptocurrency.

**Key Takeaway**

> The best wallet depends on the use case rather than the wallet itself.
