# Git Notes

## Git

Git is a distributed version control system used to track changes in code.

---

## Common Commands

git status
git add .
git commit -m "message"
git push origin main
git pull

---

## Workflow

1. Make changes
2. git status
3. git add .
4. git commit
5. git push

---

## GitHub

GitHub hosts Git repositories online for collaboration and version control.
