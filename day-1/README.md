# Day 1 – Generate Your First Solana Wallet

## Challenge

Generate my first Solana wallet using JavaScript and the Solana Kit SDK.

---

## Objective

Understand how wallets are created on Solana and learn the difference between a public key and a private key.

---

## What I Built

Using JavaScript and `@solana/kit`, I generated my first Solana wallet from the command line. The program created a unique keypair consisting of a public wallet address and a private key.

---

## What I Learned

- A Solana wallet is a cryptographic keypair.
- Every wallet has a unique public address.
- The private key proves ownership and signs transactions.
- Wallet generation happens locally on my computer.
- Anyone can know my public address, but my private key must remain secret.

---
